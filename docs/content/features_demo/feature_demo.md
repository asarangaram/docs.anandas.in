## Tabs in a page
If there are more than one proceedure, it can be given in tabs

```
=== "Procedure 1"

    Steps for procedure 1 
    
=== "Procedure 1"

    Steps for Proceedure 2

```
### Output

=== "Procedure 1"

    Steps for procedure 1 
    
=== "Procedure 1"

    Steps for Proceedure 2
